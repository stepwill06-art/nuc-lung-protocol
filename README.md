# Nuclear Medicine Lung Protocol Workflow

A comprehensive, interactive guide to the Ventilation/Perfusion (V/Q) lung imaging procedure, with separate resources for healthcare providers and patients.

## 📋 Overview

This project contains two interconnected web pages documenting the nuclear medicine lung V/Q imaging protocol:

- **Clinical Workflow** (`index.html`) - Detailed procedure guide for technicians, radiologists, and medical professionals
- **Patient Guide** (`patient.html`) - Patient-friendly explanation of what to expect

## 🎯 Features

### Clinical Workflow (index.html)
- ✅ Comprehensive protocol documentation with phase-based organization
- 🎨 Color-coded sections (Pre-Procedure, Ventilation, Perfusion, Post-Procedure)
- 📋 Interactive checklists for each phase
- ⚙️ Equipment and radiopharmaceutical specifications
- 🔍 Image acquisition protocols with timing and count requirements
- 💡 Quality assurance guidelines
- 🖨️ Print-friendly for reference cards

### Patient Guide (patient.html)
- ✅ Patient-friendly language (no medical jargon)
- 🕐 Step-by-step timeline of what happens
- 🛡️ Radiation safety information
- ❓ Comprehensive FAQ with expandable answers
- 📦 Preparation checklist
- 🎯 Clear explanation of why the test is needed
- 📱 Mobile-responsive design
- 🖨️ Print-friendly for handouts

## 📖 Usage

### For Healthcare Providers
- Use `index.html` as a reference guide during patient procedures
- Print sections for staff training materials
- Use checklists for quality assurance
- Reference equipment and radiopharmaceutical specifications

### For Patients
- Share `patient.html` link with patients before their appointment
- Print as a handout for waiting areas
- Direct patients to read before coming for their procedure
- Provide as FAQ reference
